JavaScript

JavaScript(JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages. many non-browser environment also use it, such as Node.Js, Apache CounchDB and Adobe Acrobat. JavaScript is a prototype-based, multi paradigm, single-threaded, dynamic language, supporting object-oriented, imperative and declarative(e.g funcrional programing) styles.

The standards for JavaScript are the ECMAScript Language Specification(ECMA-262)and the ECMAScript internationalization API Specification(ECMA-402)
